Project Overview :
GlowWear is a MERN Stack Fashion E-Commerce website developed using React, Node.js, Express.js, and MongoDB Atlas. The application allows users to register, log in, browse products, manage a wishlist and shopping cart, place orders, and view their profile and dashboard.

Features:

1.User Authentication
2.User Registration
3.User Login
4.JWT Authentication
5.Protected Routes

Shopping Features:

1. Product Listing
2.Product Categories
3.Wishlist Management
4.Add to Cart
5.Remove from Cart
6.Quantity Management
7.Checkout System

User Dashboard:

1.User Profile
2.Order Tracking
3.Dashboard Overview
4.Coupon & Discount Section

UI Features:

1.Responsive Design
2.Tailwind CSS Styling
3.Modern Fashion Store Interface
4.Search Bar
5.Navigation Menu
